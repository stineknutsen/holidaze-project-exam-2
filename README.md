# Holidaze venue booking

Holidaze is a holiday booking website built with React. It was built for the Noroff Front-End Development Project Exam 2. It is hosted at https://holidaze-stineknutsen.netlify.app/

## Features

- Register Page: Register as a customer or venue manager
- Login Page: Log in as a customer or venue manager
- Profile Page: can be edited. Shows customers their bookings, and managers the venues they manage and their bookings.
- Home page: search for venue, click to view venue
- Venue page: view more venue details, check availability, place a booking
- Responsive design

## Technologies used

- React
- React Context
- React styled-components
- React Hook Form
- React Router
- Noroff API
- Netlify

## Getting Started

1. Clone this repository

git clone https://github.com/stineknutsen/holidaze-project-exam-2

2.

cd holidaze

3. Install dependencies

npm install

4. Run development server

npm run dev
